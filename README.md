# Kaveesh Pro - Web-Based Photo Editor

Welcome to **Kaveesh Pro**, a high-fidelity, premium, browser-based photo editing application. It runs entirely client-side using HTML5 Canvas, Vanilla CSS (dark theme, glassmorphism, responsive panels), and Object-Oriented Vanilla JavaScript.

No server setup or package installation is required. You can run it immediately!

## 🚀 How to Run the App

1. Locate the project folder at:
   `C:\Users\User\.gemini\antigravity\scratch\pixora-pro\`
2. Double-click the **[index.html](file:///C:/Users/User/.gemini/antigravity/scratch/pixora-pro/index.html)** file, or right-click it and choose **Open with** (Google Chrome, Microsoft Edge, Firefox, or Safari).
3. The editor will boot up instantly in your web browser.

## 🎨 Features & Capabilities

* **Layer System**: Add, delete, duplicate, hide, and rename layers. Adjust layer opacity and blend modes (e.g., Multiply, Screen, Overlay, Difference) in real-time.
* **Transform Tool (Move)**: Click and drag layers to move them. Displays active bounding boxes with 8 resizing handle dots and a rotation handle, letting you scale and spin layers dynamically.
* **Drawing Brush & Eraser**: Custom paint brushes with adjustable sizes, opacities, and softness.
* **Shape Tool**: Drag-to-draw rectangles and ellipses with customizable fills, outline weights, and colors.
* **Text Tool**: Dynamic vector text layers that automatically resize their bounds based on text size, weight, and font family (supporting premium fonts).
* **Eyedropper**: Sample color codes directly from any pixel coordinate on the canvas composition.
* **Non-Destructive Adjustments**: Sliders for Brightness, Contrast, Saturation, Hue-Rotate, Blur, Grayscale, Sepia, and Invert. Switch between layers to adjust values independently.
* **Navigator Panel**: Zoom in, zoom out, fit screen, and pan (spacebar + drag) your canvas.
* **Undo & Redo History**: History panel keeps a log of actions. Go backward and forward in time by clicking on any item.
* **Export**: Save your complete layer composition merged as a high-quality **PNG** or **JPEG** file.
